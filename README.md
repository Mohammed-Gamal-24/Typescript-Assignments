# Typescript
TypeScript is a statically‑typed superset of JavaScript assignments that compiles to clean, standard JavaScript code and helps catch bugs early through type checking. This repository contains my personal TypeScript practice and small projects that I’ve built to improve my understanding of typed programming and modern development workflows.

✨ What’s in This Repo:

A collection of TypeScript files and examples demonstrating different concepts.

Practice projects showing how types, interfaces, classes, and modules work in TypeScript.

Code samples that illustrate how to write safer and more maintainable JavaScript using types.

Helpful configuration files (like tsconfig.json) for compiling TS into JavaScript.

📁 Repository Structure:

src/ — TypeScript source code.

dist/ or compiled output (if included).

tsconfig.json — TypeScript configuration.

Other practice folders and assignment files.

🚀 Project Goal:
To learn and become more comfortable with TypeScript syntax, type safety, and development tooling — building a stronger foundation for future front‑end and full‑stack applications.
